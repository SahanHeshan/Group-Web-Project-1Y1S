Introduction to CSS

              CSS Is abreviation for Cascading Style sheets
              CSS describes how HTML elements are to be displayed on the screen. It is used to control the style of a web document
                  A CSS style rule is made up of 3 parts
Selector
HTML element to which the style is applied
Property
A property is type of attiribute of HTML tag
Values
Values are assigned to propertise

There are 3 types of inseting a syle sheets
  • External CSS
  • Internal CSS
  • Inline CSS

External CSS

External style sheet can be written in any text editor and must be saved with .css extension
The external CSS file should not have any HTML tag
To link the external CSS file to HTML file we can use the <link></link> tag

Example
                       <link>rel="stylesheet" href="mystyle.css"</link>

Internal CSS

The CSS elements are placed in <style><style> tag. The style tag is placed int the head tag

example
                  
                    <head>
                       <style>
                           body{
                            background-color:pink;
                            
                           }
                           h1{
                            color:maroon;
                            margin-left:40px;

                           }
                       </style>
                    <head>
                    <body>
                       <h1>This is a heading</h1>
                       <p>This is a paragraph<p>
                    <body>

Inline CSS

This is CSS style affecting to a particular line
Example 

                  
                    <body>
                    <h1 style="color:blue; text-align:center;">
                    < style="color:red;">
                    </body>
                    <html>

Style Precedence

  1. Inline Style
  2. External and Internal style sheets
  3. Browser default

Property in CSS

CSS property is used to style the HTML elements. Each html element can be styled using different CSS propertise.
There are many CSS propertise to style the elements of HTML.

  • color
  • Size
  • border
  • Text
A CSS property consits of a property name and property value. To specify more than one CSS property each pair is seperated
by a semicolon

Text Property of CSS

Font Family
Allow us to change the font type used .Any standard font is used
Font Size
May be specified using a few different ways.
Font Weight
May be specified using a few different ways

Color Property Of CSS

color
Allows to specify the color of the text
Backgroung Color
Allows us to specify the background color of an element

Spacing Property of CSS

padding
Refers to the space around our content
Margin
Similar to padding, That affects the spacing aroung the content
Border
Is in between the padding and margin
